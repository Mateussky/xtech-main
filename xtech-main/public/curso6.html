Responda as perguntas a seguir:


Git é um sistema de controle de versão distribuído usado para rastrear mudanças no código fonte durante o desenvolvimento de software.
Git é um programa de edição de texto usado para escrever código.
Git é uma ferramenta para gerenciar banco de dados.


Você pode usar o comando git clone seguido da URL do repositório: git clone https://github.com/usuario/repositorio.git.
Você deve usar o comando git copy seguido do caminho do repositório.
Para clonar um repositório, você precisa baixar um arquivo ZIP do GitHub.


Um commit é uma alteração registrada no repositório. Para fazer um commit, você deve usar os comandos git add . para adicionar as mudanças e git commit -m "mensagem" para registrar.
Um commit é um tipo de backup. Você pode fazer um usando o comando git backup.
Commits são usados para apagar arquivos do repositório.


Branches são ramificações do código que permitem desenvolver funcionalidades de forma isolada. Elas ajudam a manter o código organizado e facilitam o trabalho em equipe.
Branches são cópias do repositório que não podem ser alteradas.
Branches são usados para organizar a documentação do projeto.


Conflitos ocorrem quando duas branches têm mudanças incompatíveis. Para resolver, você deve editar os arquivos em conflito, selecionar as alterações que deseja manter e depois realizar um commit.
Você pode simplesmente descartar todas as mudanças e continuar com o merge.
Conflitos são resolvidos automaticamente pelo Git e não precisam de intervenção.
